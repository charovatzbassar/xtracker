# XTracker
### Tracking budget on the go!

>Welcome to XTracker, your solution for budget tracking. Users can add expenses, income and savings on the go, and view total expenses on-demand.
>XTracker provides seamless user experience, and a robust design which utilises cutting-edge technologies.<br><br>
>This solution was created using Android Jetpack Compose, further accompanied with Room database and Navigation Compose library.<br><br>
>XTracker was created as a student project for course <b>IT 206 - Introduction to Mobile Programming</b>, which carries <b>25%</b> of the total course grade.<br><br>
